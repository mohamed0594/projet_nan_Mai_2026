 Projet Immobilier - Site de Vente et Location
 Description de mon site web as sorbr ....

Ce projet est un site web immobilier permettant de consulter, rechercher et explorer des biens immobiliers tels que des maisons, villas, appartements et résidences.

L’objectif est de proposer une interface moderne, simple et intuitive pour faciliter la recherche de logements.


Fonctionnalités

- Affichage des biens immobiliers
- Fiches détaillées (prix, ville, caractéristiques)
- Barre de recherche de logements
- Interface utilisateur moderne
- Formulaire de recherche personnalisé
- Design responsive (adapté aux écrans)

Technologies utilisées

- HTML5
- CSS3
- Font Awesome (icônes)
- Git & GitHub
